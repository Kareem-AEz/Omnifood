# Omnifood

Omnifood is a course completion project created by Jonas Schmedtmann. This project showcases the skills and techniques learned in the Intermediate HTML and CSS course, focusing on creating a modern and responsive web design.

Check out the live version here: [Omnifood on Netlify](https://omnifood-spaghetti.netlify.app/)

## Current Progress

- **Intermediate HTML & CSS:** This is the foundation of the project, demonstrating a solid understanding of HTML5 and CSS3 principles.

## Next to Come

- **Advanced CSS and SASS:** Enhancing styling capabilities with advanced techniques and SASS for better CSS management.
- **The Complete JavaScript Course: From Zero to Expert:** Adding interactivity and dynamic features using JavaScript.
- **The Ultimate React Course:** Integrating modern frameworks like React, Next.js, and Redux for a seamless user experience.
- **Backend Development:** Learning Node.js, Express, and MongoDB for server-side programming and database management.
